# Changelog

All notable changes to the FPolicy Add-on for Splunk are documented in this file.

## [1.8.1] - 2026-01-14

### Added
- SSL/TLS encryption support with TLS 1.2
- Certificate upload functionality for secure connections
- Server-side SSL certificate and private key configuration

### Removed
- Deprecated Accounts section

## [1.7.6] - 2024-05-09

### Added
- Sourcetype selection functionality
- Support for up to 8 nodes per cluster

### Fixed
- Resolved intermittent connection issues

## [1.5.6] - 2024-04-11

### Added
- Full cluster support for multi-node ONTAP environments

## [1.3.3] - 2024-03-28

### Added
- Single SVM node support

## [0.7.1] - 2023-12-20

### Changed
- Enhanced logging functionality (`splunk/var/log/splunk/server_input.log`)

## [0.6.5] - 2023-10-20

### Added
- Initial tests with cloud instance completed
- First production-ready release
